数値と文字列には違いがあることを知ろう

数字はそのまま半角数字で記述。例:1

文字列はクォートで囲む。例:'1'

実践~irbで確かめてみよう~

『1.class』と入力し実行すると『integer』(数値)と表示される。

『'1'.class』と入力し実行すると『string』(文字列)と表示される。

pメソッドとは

pメソッドとputsメソッドは引数のデータの種類を考慮してデータを返す(表示する)点で異なっている。

例:putsメソッドでは引数に１を指定しても'1'を指定しても同じように『1』と表示されnilが返ってきた。
[単に画面に表示して改行する用途で使用。]

例:pメソッドでは1なら1、'1'なら'1'と区別して表示され、返ってきたデータも異なった。
[主にテスト用途で使用。データの種類がわかりやすい。]

データの種類

データ型:データの種類のこと。

Rubyでそれぞれのデータのデータ型を知りたいときは、データの後に.classと続けて書くことで確認できる。

例 種類 データ型名
10 整数値 Integer(古いRubyではFixnum)
20.1 浮動小数点数 Float
'matz' 文字列 String
String クラス Class
true 真偽値 True Class/False Class
{} ハッシュ Hash
[] 配列 Array
nil ニル NilClass
